학교 시간표 프로그램 — Windows 사용 안내
========================================

【 설치 】
1. ZIP 파일을 원하는 폴더에 압축 해제합니다. (예: C:\Timetable)
2. 폴더 안의 "시작.bat"을 더블클릭합니다.
3. 브라우저가 자동으로 열리면 사용을 시작할 수 있습니다.

【 종료 】
- 검은색 명령 창을 닫으면 프로그램이 종료됩니다.

【 데이터 】
- 각 PC의 시간표 데이터는 "data" 폴더의 timetable.db 파일에 저장됩니다.
- PC마다 처음 실행 시 빈 데이터베이스가 자동으로 만들어집니다.
- data 폴더를 삭제하면 처음부터 다시 시작할 수 있습니다.

【 문제 해결 】
- 방화벽에서 Node.js 또는 이 프로그램의 로컬 연결을 허용해 주세요.
- 3000번 포트를 다른 프로그램이 사용 중이면 해당 프로그램을 종료한 뒤 다시 시도하세요.
- Windows Defender가 차단하면 "추가 정보" → "실행"을 선택하세요.

【 인터넷 】
- 프로그램 실행 자체는 오프라인에서도 가능합니다.
- 학사일정 공휴일 자동 불러오기는 API 키 설정 시에만 인터넷이 필요합니다.
